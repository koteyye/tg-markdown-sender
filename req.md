**Ну всё, Telegram наконец-то завёз Markdown 🎉**

Теперь посты можно нормально оформлять прямо через разметку: **жирный текст**, *курсив*, `код`, ссылки, списки и всё остальное без ручного тыканья по редактору.

### Что теперь доступно:

* ✅ Маркированные списки
* 🔢 Нумерованные списки
* 💬 Цитаты
* 📌 Чекбоксы:

  * [x] Написать пост
  * [x] Добавить форматирование
  * [ ] Не забыть про мемы
* 🧩 Вставка кода:

```
console.log("Markdown рулит 🚀");
```

> И да, теперь текст выглядит намного приятнее без лишней возни.

Можно даже делать таблицы:

| Фича         | Работает |
| ------------ | -------- |
| Жирный текст | ✅        |
| Курсив       | ✅        |
| Код          | ✅        |
| Мемный вайб  | 😎       |

А значит, посты от нейронок теперь будут писаться гораздо проще: попросил модель сразу выдать текст в Markdown, скопировал, вставил — и оформление уже на месте.

Больше не надо отдельно выделять заголовки, вручную расставлять ссылки и полировать каждый абзац после генерации.

Мелочь, конечно, но для тех, кто регулярно генерирует контент через ИИ, — прям очень приятное обновление.

**Нейронка написала → вставил в Telegram → опубликовал.**
Контент на коленке стал ещё быстрее 😄
